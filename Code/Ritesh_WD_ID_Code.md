- ## Case 1 :pushpin:
use of Id Selector :pencil2:

``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ID Selector</title>
    <style>

            #green{
                color:green;
            }

    </style>
</head>
<body>

    <div>
        <p id="green"> Hello I am Ritesh </p>
    </div>
    
</body>
</html>
```
